## 🔄 System Flow

```mermaid
flowchart TD
A[Register] --> B[Enter Info]
B --> C[Select Mess Name]
C --> D[Enter Secret Code]
D --> E[Join System]
E --> F[Meal ON/OFF]
F --> G[Manager Adds Cost]
G --> H[Deposit Added]
H --> I[Dashboard Analytics]
```

## ⚙️ Installation

```bash

cd smartmeal
npm install
npm run dev
```
